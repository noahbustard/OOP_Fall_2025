## Overview

This repo is being used to:

- organize my OOP assignments in one place
- track progress across the semester
- document build steps, dependencies, and outputs
- showcase work in C++ with SDL2 and JSON-driven programs

The current work in this repository is centered around building the early foundation for a **Game of Life** project pipeline using **C++**, **SDL2**, and **JSON**.
